Detailed Description

template<typename Dim_, typename Index_, typename Float_>
class knncolle::Prebuilt< Dim_, Index_, Float_ >

Interface for prebuilt nearest-neighbor search indices.

Template Parameters
Dim_Integer type for the number of dimensions. For the output of Builder::build, this is set to MockMatrix::dimension_type.
Index_Integer type for the indices. For the output of Builder::build, this is set to MockMatrix::index_type.
Float_Floating point type for the query data and output distances.
