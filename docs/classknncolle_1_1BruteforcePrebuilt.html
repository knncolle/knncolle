Detailed Description

template<class Distance_, typename Dim_, typename Index_, typename Store_, typename Float_>
class knncolle::BruteforcePrebuilt< Distance_, Dim_, Index_, Store_, Float_ >

Index for a brute-force nearest neighbor search.

Instances of this class are usually constructed using BruteforceBuilder::build.

Template Parameters
Distance_A distance calculation class satisfying the MockDistance contract.
Dim_Integer type for the number of dimensions. For the output of BruteforceBuilder::build, this is set to MockMatrix::dimension_type.
Index_Integer type for the indices. For the output of BruteforceBuilder::build, this is set to MockMatrix::index_type.
Store_Floating point type for the stored data. For the output of BruteforceBuilder::build, this is set to MockMatrix::data_type. This may be set to a lower-precision type than Float_ to save memory.
Float_Floating point type for the query data and output distances.
